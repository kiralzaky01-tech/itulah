KEMAMPUAN SAYA DI BIDANG PEMROGRAMAN

Ini adalah kemampuan saya dalam membuat sebuah program menggunakan berbagai bahasa pemrograman.

Python

Python adalah bahasa pemrograman yang populer dan mudah dipelajari oleh pemula. Python sering digunakan untuk membuat website, aplikasi, kecerdasan buatan (AI), data science, otomatisasi, hingga cyber security. Bahasa pemrograman ini memiliki sintaks yang sederhana sehingga mudah dibaca dan dipahami.

HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur sebuah website. HTML berfungsi untuk mengatur isi halaman web seperti judul, paragraf, gambar, tombol, link, dan berbagai elemen lainnya agar dapat ditampilkan di browser.

CSS

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan desain sebuah website. CSS berfungsi untuk mempercantik halaman web yang dibuat menggunakan HTML, seperti mengatur warna, ukuran teks, posisi gambar, layout, animasi, hingga membuat tampilan website menjadi responsif di berbagai perangkat.
